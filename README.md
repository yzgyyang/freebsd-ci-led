# freebsd-ci-led
An LED panel indicating the FreeBSD Jenkins CI build status, backed by FreeBSD on an IoT device.  

## Requirements:
A BeagleBone Green running FreeBSD 12.0-CURRENT and python 2.7  
GPIO connection to LEDs at /dev/gpioc0
